Welcome to IST 4035

Advanced Web Design and Application

Lecturer: Austin Sifuna

This course is about web design and applications.
It involves use of HTML, CSS and javascript for the front-end design.
Web applications are usually built to handle business transactions.
Given the sensitivity and volumes of business transactions, web applications need to be designed in such a way that the system can scale well, be secure, and flexible enough to easily accommodate future enhancements.
These considerations, among others, make building web applications a fairly complex undertaking.
This course explores various aspects of advanced web development, including Color Theory, Responsive Design, Advanced CSS Features, ECMAScript 6, Web Services, and JSON.
